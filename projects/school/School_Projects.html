About these projects

These are projects I crafted throughout my journey to obtaining my Bachelors in Software Engineering. While in school, I had the pleasure of coding in backend scripts such as Python3, Java, and SQL. I also have experience in front-end development using CSS, HTML, and JavaScript. Below you can find these projects so enjoy!

Memory Game

This project was a lot of fun to complete and I learned alot about Python! This is a small game where you pick two cards and see if they match. If they do, they automatically clear themselves from the board. It has settings for difficulty and will remember those settings on next startup.
